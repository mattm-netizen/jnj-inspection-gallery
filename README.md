# J&J Equipment Rental - Inspection Photo Gallery

Public inspection photo gallery for PDI (Pre-Delivery Inspection) and RDI (Return Delivery Inspection) photos.

## Routes

- `/contract/[contractNumber]` - View PDI photos for a contract
- `/invoice/[invoiceNumber]` - View RDI photos for an invoice

## Deployment

This project is deployed on Vercel and fetches photos from:
`https://rentalpro.jnjequipmentrental.com/api/public/inspection/`
